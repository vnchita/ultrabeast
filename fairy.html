FAIRY

Fairy type is the newest type in the game if we do not include terastalization.

Despite not hitting a lot of types for super effective damage, it is generally considered exceptionally good offensively and defensively because it is the only type that is able to hit powerful Dragon type Pokèmon and is also completely immune to Dragon type moves.

Attack pros & cons

Fairy type moves are super effective against:

FIGHTING
DARK
DRAGON

Fairy type moves are not very effective against:

STEEL
POISON
FIRE

Defense pros & cons

Fairy type Pokèmon resist:

FIGHTING
BUG
DARK

Fairy type Pokèmon are weak against:

POISON
STEEL

Fairy type Pokèmon are immune to:

DRAGON

There are no recorded Ultrabeasts with this type.
